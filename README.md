# 货币时代：平遥

## 1 游戏概述

### 1.1 背景设定

**货币时代：平遥** 是一款以清代中晚期中国商业发展为背景的经济策略桌游，灵感来源于《工业时代：伯明翰》的游戏机制。游戏时间跨度为1750年至1911年，玩家将扮演平遥商帮的掌柜，通过经营茶叶、瓷器、丝绸、白银等商品，建立票号网络，最终成为商界巨擘。

### 1.2 游戏特色

- **动态时代演进**：游戏分为两个时代——乾隆盛世与晚清变局，每个时代有不同的商业机会和挑战
- **资源网络建设**：建立商路网络，连接产地、集散地与消费市场
- **票号金融系统**：通过票号进行资金流转，掌控清代金融命脉
- **历史事件影响**：洋务运动、太平天国、鸦片战争等事件将改变商业格局

- **白银**：通用货币，也是可交易商品
- **茶叶**：分为绿茶、红茶、砖茶三种等级
- **瓷器**：分为粗瓷、细瓷、御瓷三个等级
- **丝绸**：分为生丝、织绸、刺绣
- **票号**：特殊资源，代表金融网络节点

### 2.3 卡牌系统

**产业卡牌**（48张）：

**市场卡牌**（36张）：

## 3 游戏机制

### 3.1 核心机制

#### 网络建设

玩家可以在地图上建设：
- **商路**：连接城市，运输货物
- **商号**：生产或加工商品
- **票号**：提供资金流转服务
- **货栈**：储存货物，等待市场机会

#### 资源循环

```
产地 → 集散地 → 加工 → 市场 → 利润
  ↓                      ↓
投入 ←───────────── 资金流转（票号）
```

### 3.2 行动系统

每位玩家回合可执行2个行动（可重复）：

| 行动 | 效果 | 限制 |
|------|------|------|
| 建设产业 | 放置/升级一个产业标记 | 需满足位置和前置条件 |
| 开发商路 | 连接两个城市，运输货物 | 需拥有运输产业 |
| 票号汇兑 | 将白银转移到任意城市 | 需拥有票号网络 |
| 市场交易 | 将货物出售给市场 | 需有足够市场需求 |
| 招募伙计 | 获得额外行动能力 | 需支付白银 |
| 结盟联姻 | 与其他玩家合作 | 双方同意 |

### 3.3 时代系统

**乾隆盛世时代（1750-1840）**：
- 重点：国内贸易、茶叶瓷器生产
- 特殊：贡品订单、官商关系
- 限制：外贸仅限十三行、恰克图

**晚清变局时代（1840-1911）**：
- 重点：票号金融、铁路运输、洋行贸易
- 特殊：铁路建设、现代工业
- 风险：战乱、通货膨胀、外资竞争

### 3.4 事件卡牌详解

事件卡牌在游戏过程中随机抽取，立即生效。以下是24张事件卡牌的详细效果：

#### 乾隆盛世时代事件（10张）

| 卡牌名称 | 触发时机 | 效果 |
|----------|----------|------|
| **乾隆盛世** | 第一时代第2回合 | 茶叶、瓷器市场需求永久+1；所有玩家获得10两白银 |
| **千叟宴** | 第一时代中期 | 拥有最高等级御窑的玩家获得宫廷贡品订单（价值30分）；其他拥有御瓷的玩家获得15分 |
| **平定准噶尔** | 第一时代 | 西北商路解锁；恰克图贸易需求+3 |
| **马戛尔尼使团** | 第一时代 | 十三行外贸需求临时+2（持续3回合）；拥有广州产业的玩家获得5两白银 |
| **和珅倒台** | 第一时代后期 | 查抄财产：拥有最多白银的玩家损失20%白银；抄没白银进入市场池 |
| **白莲教起义** | 第一时代后期 | 华中地区商路暂时断绝（2回合）；在该区域的产业无法运输货物 |
| **嘉庆改制** | 时代转换点 | 贡品订单需求-1；所有玩家支付5两白银作为"整顿费用" |
| **道光萧条** | 第二时代初期 | 所有商品市场需求-1；白银购买力下降（每10两白银按8分计算） |
| **两淮盐商案** | 第二时代初期 | 拥有最多产业的玩家随机损失一个3级以下产业 |
| **票号崛起** | 第二时代初期 | 解锁票号产业；所有玩家获得一次免费建设票号的机会 |

#### 晚清变局时代事件（14张）

| 卡牌名称 | 触发时机 | 效果 |
|----------|----------|------|
| **鸦片战争** | 第二时代早期 | 广州十三行关闭；所有外贸需求转移至上海；拥有广州产业的玩家损失10两白银 |
| **南京条约** | 鸦片战争后 | 解锁上海、宁波、福州、厦门、广州五口通商；外资洋行进入（每回合消耗市场需求1点） |
| **太平天国** | 第二时代中期 | 江南地区（苏州、杭州、扬州）所有产业暂时停运（3回合）；该区域商路被破坏 |
| **湘军崛起** | 太平天国后 | 拥有华中等产业的玩家可选择支付15两白银保护产业；未保护的产业永久损毁 |
| **第二次鸦片战争** | 第二时代 | 北京至天津商路断绝；圆明园被焚（拥有御瓷的玩家损失20%相关收益） |
| **洋务运动** | 第二时代中期 | 解锁铁路、电报、纺织厂等现代产业；玩家可花费25两白银升级一个产业为现代企业 |
| **左宗棠收新疆** | 第二时代 | 西北商路恢复；恰克图贸易需求+2 |
| **中法战争** | 第二时代 | 西南地区商路暂时断绝（2回合）；广西、云南产业受影响 |
| **甲午战争** | 第二时代后期 | 所有玩家损失30两白银作为"赔款"；日本商品进入市场（竞争需求） |
| **马关条约** | 甲午战争后 | 解锁重庆、苏州、杭州、沙市为通商口岸；外资需求+3但本土需求-2 |
| **戊戌变法** | 第二时代 | 所有玩家可选择支付20两白银获得"维新派"标记（时代结束+8分）；不支付则无效果 |
| **义和团运动** | 第二时代后期 | 华北地区产业暂时停运（2回合）；所有票号无法进行汇兑业务 |
| **庚子国变** | 义和团后 | 所有玩家再次损失40两白银作为"庚子赔款"；北京产业永久损毁50% |
| **清末新政** | 第二时代末期 | 铁路建设加速；票号需求+3；所有玩家最后计分时白银购买力恢复（1:1） |

#### 事件卡牌特殊规则

1. **事件堆叠**：同类事件同时触发时，效果叠加（如多次赔款）
2. **区域影响**：区域事件仅影响该区域内玩家放置的产业和商路
3. **豁免机制**：拥有"官商关系"标记的玩家可豁免一次负面事件
4. **事件选择**：当事件牌生效时，玩家可选择是否支付白银避免负面影响
5. **历史遗产奖励**：经历过5个以上重大历史事件并存活至游戏结束的玩家，获得"历尽沧桑"成就+5分

## 4 计分规则

### 4.1 时代结束计分

每个时代结束时计算：
1. **产业得分**：所有产业等级之和
2. **网络得分**：连通的城市数量 × 2
3. **白银储备**：每10两白银 = 1分
4. **票号网络**：票号覆盖城市数量 × 3

### 4.2 最终计分

游戏结束时额外获得：
1. **市场支配**：每种商品的最大市场份额奖励
2. **票号霸主**：拥有最多票号的玩家 +15分
3. **商路之王**：连接最多城市的玩家 +10分
4. **历史遗产**：完成特殊成就的玩家 +5-10分

## 5 策略要点

### 5.1 早期策略

- 专注单一商品产业链（如茶叶或瓷器）
- 建立区域运输网络
- 积累白银储备

### 5.2 中期策略

- 扩展票号网络，提升资金流动性
- 多元化商品组合，分散风险
- 抓住时代转换的机会

### 5.3 后期策略

- 争夺高价值市场份额
- 利用铁路和票号建立全国网络
- 应对历史事件，保持资产稳定

## 6 扩展内容

### 6.1 可扩展模块

- **海上丝绸之路**：增加沿海贸易和海运机制
- **西域商道**：扩展中亚贸易路线
- **南洋开发**：东南亚华人商业网络
- **票号大战**：专注金融系统的二人对抗模式

### 6.2 变体规则

- **合作模式**：玩家组成商帮，共同对抗外资入侵
- **历史剧本**：特定历史时期的固定开局
- **短局模式**：仅玩一个时代（约60分钟）

---

**设计说明**：本游戏设计旨在通过桌游形式展现清代中国商业经济的历史变迁，让玩家在策略博弈中感受晋商票号的兴衰荣辱。游戏机制借鉴了《工业时代：伯明翰》的网络建设和市场动态系统，但融入了中国特有的票号金融和商帮文化元素。
